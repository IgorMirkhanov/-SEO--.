MarPla - SEO Product Description Generator

Финальная передача заказчику (One Pager)

NestJS React + Vite Telegram Web App Flowise-ready SSE Streaming

Цель

Сервис генерации SEO-описаний товаров для e-commerce с потоковой выдачей, строгой валидацией структуры и устойчивостью к ошибкам AI.

Что реализовано

  • Форма ввода: product_name, category, keywords.
  • Генерация структуры: title, meta_description, h1, description, bullets[].
  • Endpoint: POST /api/generate-seo (JSON).
  • Endpoint: POST /api/generate-seo/stream (SSE: seo-token, seo-done, seo-fallback).

Надежность

  • Timeout внешнего AI-запроса.
  • DTO validation + JSON schema validation.
  • Graceful fallback при пустом/битом ответе.
  • Логирование этапов и CI-проверка сборки.

Запуск

  • Локально: npm install -> npm run dev:backend + npm run dev:frontend.
  • Docker: docker compose up --build, затем открыть http://localhost:8080.

Статус для ТЗ

Проект поставлен в demo-ready режиме: FLOWISE_MOCK_MODE=true, поэтому заказчик может сразу проверить полный сценарий без реальных API-ключей.
